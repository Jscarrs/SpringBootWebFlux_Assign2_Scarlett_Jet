# SpringBootWebFlux_Assign2_Scarlett_Jet

## Course
ITE5435 - Enterprise Application Modelling Labs

## Description
This project is a Spring Boot WebFlux application using Reactive MongoDB.

It demonstrates:
- Reactive CRUD operations using Mono and Flux
- Inheritance mapping (Account, Checking, Saving)
- MongoDB persistence
- REST API testing with Postman

## Technologies Used
- Spring Boot 3
- Spring WebFlux
- Reactive MongoDB
- Lombok
- MongoDB Compass
- Postman

## Database Configuration
- Database Name: humber
- Port: 27017
- Server Port: 8084

## Endpoints

### Account
- POST /accounts
- GET /accounts
- PUT /accounts/{id}
- DELETE /accounts/{id}

### Checking
- POST /checking
- GET /checking

### Saving
- POST /saving
- GET /saving
